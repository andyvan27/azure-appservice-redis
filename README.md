# azure-appservice-redis
 How Azure Cache for Redis scales Azure App Services

 ## Start local Redis on Docker Desktop
 `docker compose up -d`

 ## Check Redis container
 ```
 # redis-cli            
127.0.0.1:6379> ping
PONG
127.0.0.1:6379>
 ```
